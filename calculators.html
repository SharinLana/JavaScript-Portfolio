Calculators / Converters

Roommate Calculator

Features:

  • Summation of the monthly income of spouses or two roommates
  • Summation of all monthly expenses
  • Calculate the payout share of each spouse/roommate based on a percentage of their monthly income
  • Family finance assessment

App basis:

  • ✓ JavaScript + SweetAlert (methods: Math.floor(), Number(), toFixed(), forEach(), etc.)
  • ✓ HTML/CSS (Flexbox, CSS animation, responsive web design, etc.)

"Go Dutch" Calculator

Features:

  • Splitting the bill equally among all guests
  • Adding tips at the discretion of guests
  • Ability to enter custom tips

App basis:

  • ✓ JavaScript + SweetAlert (arithmetic operations)
  • ✓ HTML/CSS (Flexbox, CSS animation, responsive web design, etc.)

Recipe Search

Features:

  • Search for dishes by name
  • Two language versions of the application
  • Built-in kitchen timer with pause option
  • Built-in mass and volume converter

App basis:

  • ✓ JavaScript + SweetAlert (methods: toLowerCase(), includes(), forEach(), toFixed(), setAttribute(), setInterval(), Math.floor(), isNaN(), modulus operator (%), ternary oparetor, etc.)
  • ✓ HTML/CSS (Flexbox, responsive web design, etc)

Currency Converter API

Features:

  • Connecting to a database of current exchange rates using an api key
  • Currency convertion

App basis:

  • ✓ JavaScript + JSON API + SweetAlert (methods: async - await, json(), isNaN(), toFixed(), arithmetic operations, etc.)
  • ✓ HTML/CSS (Flexbox, CSS animation, responsive web design)
